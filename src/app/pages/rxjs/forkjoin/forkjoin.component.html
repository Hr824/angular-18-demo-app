RxJS : Opérateur forkjoin

forkJoin attend que toutes les observables aient émis une valeur, puis il combine ces valeurs en une seule émission.
Ici, 2 observables : 1 liste de livres, 1 liste d'auteurs, jointure sur l'id de l'auteur
@defer { @for(book of booksWithAuthors$ | async; track book.id){ }
Id Livre Titre Auteur
{{ book.id }} {{ book.title }} {{ book.author?.firstname }} {{ book.author?.lastname }}
